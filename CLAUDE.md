# Claude Code Configuration

## Project: vocab-cat-test

### Development Commands

- `npm run dev` - Start development server
- `npm run build` - Build the application
- `npm run test` - Run tests
- `npm run lint` - Run linting
- `npm run typecheck` - Run TypeScript type checking

### Key Information

- This is a vocabulary assessment application using IRT (Item Response Theory)
- Main components are in `irt_cat_engine/` directory
- Uses Docker for containerization
- Deployed on both Vercel (frontend) and Google Cloud Run (backend)

### Testing

Run local tests using the guide in `LOCAL_TESTING_GUIDE.md`

### Notes

- Check `DEPLOYMENT_STATUS.md` for current deployment information
- Use `docker-compose.yml` for local development with databases
